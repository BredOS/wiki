# 🔁 3. 替换面板图形

- 用标准的`mesa`软件包替换`mesa-panfork-git`软件包：

```
sudo pacman -S mesa
```

# 🔁 4. 重启您的系统

- 安装vulkan加载器和驱动器：

```
sudo pacman -S vulkan-icd-loader vulkan-panfrost
```

# 5. 重启您的系统

- 重启系统以应用更改。 如果您想要验证您的图形，您可以运行以下操作：

```
sudo pacman -S inxi mesa-utils
inxi -G
```
